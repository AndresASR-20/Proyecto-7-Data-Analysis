# Proyecto-7-Data-Analysis
Proyecto 7 Data Analysis
Consiste en una aplicación pensada para ser expuesta mediante Render, que consiste en un graficador que puede generar un histograma de la columna "odometer" o un gráfico de dispersión entre las columnas "odometer" y "price", provenientes del conjunto de datos "vehicles_us". La app está hecha de tal forma que el ususario pueda decidir si se generará solo el historgrama, solo la gráfica de dispersión, o ambos.
